# mettingroom

## 사용라이브러리/툴
- springboot 2.0.3
- h2
- jpa

## 서버 실행 방법
- git clone https://github.com/cookker/meetingroom.git
- gradle build
- java -jar ./build/libs/meetingroom-0.0.1-SNAPSHOT.jar
- 웹브라우저에서 http://localhost:8080 접속
